About

Vanity remote import paths (as they are called in the Golang reference) have been implemented for a while, but do not seem to be used that often. I assume this is because you need to maintain an HTML header section. This is what ImportAlias will do for you.

Quickstart

  1. Login: I tried to keep the overhead as small as possible. You don't need to remember any credentials, I only offer 3rd party login anyway. Right now I support GitHub and Google. Just click the Login button in the top right corner.

  2. Domain claiming: Before you can add an alias, you need to claim a domain. Right now a domain can be managed by one person (one account) only. The architecture has been designed to support delegates, but the functionality has not been exposed just yet.

    To verify that you are actually the owner of the domain you are about to claim, the CNAME record for the forwarding has to be prefixed by your User ID (UID). The check is made against Google’s DNS and not resolved manually so give the record some time to propagate. You can find your UID (as well as your personal API key) in the dropdown in the navigation bar at the top.

    Example: Let’s assume you want to claim the domain go.example.com. Go to your DNS provider and create a CNAME record for [your UID].go.example.com and let it point to importalias.surmair.de.

  3. Aliases: Now that you have your domain set up you can add aliases. For each alias, the following information is required:

    Alias
    The path which should be aliased for a remote repository. It may contain multiples slashes.
    Repository
    Type and URL of said repository. At the moment Git, Mercurial and Bazaar are supported by go get.
    Forward URL
    The URL to forward to when it is accessed by something else than go get (e.g. a browser).

Alpha

Important: I give this service the “alpha stage” mark. Nothing has been tested in the wild and I will not give any guarantees. If you rely on your Go imports and need them to work 24/7, do not use this service just yet. On the contrary: I want you to try to break it, do weird stuff, use the API (*gasp*).
